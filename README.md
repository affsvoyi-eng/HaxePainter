# HaxePainter

State-based drawing app made with:
- HaxeFlixel
- OpenFL
- Lime

Features:
- Desktop support
- Mobile support
- PNG export
- Drawing system
- Brush size
- Color palette
- Text system
- Custom canvas size
- Background image support

Background:
assets/images/bg/bg.png

Run:
lime test windows
lime test linux
lime test android
